# github-final-project

This is a simple interest calculator project for the Coursera course "Getting Started with Git and GitHub".

## Description

This project contains a simple interest calculator script written in Bash. The script calculates the simple interest based on principal amount, rate of interest, and time period.

## Features

- Calculates simple interest
- Takes user input for principal, rate, and time
- Displays the calculated interest

## Usage

Run the script using the following command:

```bash
./simple-interest.sh
```
